# Randomizer API

## Requirements

NodeJS (v14)
npm (package manager)

## Installation

```bash
$ npm install
```

## Development Setup
```bash
1. Create `.env` file. (You can copy `.env.example` file)
4. Run `npm install` to install all the dependencies.
5. Run `npm start` or `npm start:dev` to start the server.
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ yarn start:dev

# production mode
$ npm start:prod
```
## Test

```bash
# unit tests
$ npm test

# e2e tests
$ npm test:e2e
```
